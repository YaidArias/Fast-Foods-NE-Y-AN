# 🍖 Comidas Rápidas NE&AN - App de Pedidos v2 (Con Firebase)

Aplicación web progresiva (PWA) para recibir pedidos con base de datos en tiempo real.

## ✨ Características Nuevas

- 📱 **App para clientes** - Menú, carrito, checkout integrado
- 🔥 **Firebase Firestore** - Pedidos guardados en tiempo real
- 📊 **Panel de administración** - Ve todos los pedidos, cambia estados
- 🔔 **Notificación WhatsApp** - Te avisa cuando llega un pedido nuevo
- 👤 **Mis Pedidos** - Clientes ven el estado de sus pedidos
- 🏷️ **Estados del pedido** - Nuevo → En preparación → Listo → Entregado

## 📋 Configuración de Firebase

1. Crea proyecto en [firebase.google.com](https://firebase.google.com)
2. Activa Firestore Database (modo prueba)
3. Activa Authentication (Email/Password)
4. Copia tu configuración en `index.html`

## 🔐 Panel Admin

- URL: `tu-link.vercel.app` → botón "Panel de Administración"
- Necesitas crear usuario en Firebase Authentication

## 🚀 Deploy

```bash
git add .
git commit -m "v2 con Firebase"
git push origin master
```

Vercel se actualiza automáticamente.
